# BogoTemplate

A customizable HTML template for creating BOGO (Buy One Get One) promotional widgets. This template provides a clean, modern interface for displaying product options with pricing, discounts, and selection features.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Customizable Options**: Easily modify product details, pricing, and styling
- **Interactive Selection**: Radio button style selection with visual feedback
- **Discount Badges**: Highlight savings and popular items
- **Size/Color Selectors**: Optional variant selection for products
- **Modern UI**: Clean, professional styling with smooth transitions

## Usage

1. Clone or download the repository
2. Open `bogo-widget.html` in your browser or integrate it into your website
3. Customize the HTML content, CSS styles, and JavaScript (if any) to fit your needs

## Customization

### Changing Product Options

Edit the HTML structure within the `.widget` container to modify:
- Product names and descriptions
- Pricing information
- Discount percentages
- Available sizes/colors

### Styling

Modify the CSS variables and styles to match your brand:
- Color scheme (currently using #e8617a as primary color)
- Typography
- Spacing and layout
- Hover effects and transitions

### Adding Functionality

The template is pure HTML/CSS. To add interactivity:
- Include JavaScript for form submission
- Add validation for selections
- Integrate with your e-commerce platform
